## 📌 About

**Keen Keeper** is a fun and modern social hangout web app that brings friends closer — no matter where they are. Connect with your crew through instant messaging, voice calls, and video hangouts, all from one clean and fast interface.

Whether you want to catch up, make plans, or just vibe together, Keen Keeper is the place to be.

> *Your friends are one click away.*

---

## ⚡ Key Features

### 💬 Instant Messaging
Chat with your friends in real time. Send messages, share thoughts, and keep the conversation going — fast and smooth.

### 📞 Voice & Video Calls
Jump on a call whenever you want. Keen Keeper supports both audio and video calls, so you can see your friends face-to-face no matter the distance.

### 🎉 Friend Hangout Space
A dedicated social space designed just for hanging out. Connect, laugh, and spend time with the people who matter most.

## 🚀 Getting Started

### Prerequisites
- Node.js `v18+`
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/sahidul-dev-47/Keen-Keeper-Web.git

# 2. Navigate into the project
cd Keen-Keeper-Web

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Then open your browser and visit `http://localhost:5173` 🚀
